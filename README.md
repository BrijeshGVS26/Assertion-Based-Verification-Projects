# 🔍 Assertion-Based Verification Projects (SystemVerilog)

## 📌 Overview
This repository showcases Assertion-Based Verification (ABV) of digital designs using SystemVerilog.

## 🧠 Skills
- SystemVerilog Assertions (SVA)
- Functional verification
- RTL design validation
- Bind-based verification

## 📂 Projects

### FSM Verification
- One-hot FSM
- Verified state transitions and reset behavior

### Adder Verification
- 4-bit adder
- Verified using bind assertions

### Counter Verification
- Up/Down counter
- Verified increment/decrement and reset logic

### FIFO Verification
- FIFO with read/write operations
- Verified full/empty conditions and control logic

## 🛠 Tools
- SystemVerilog
- Simulation-based verification
